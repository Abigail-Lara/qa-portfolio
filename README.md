# Portafolio de QA – Patricia Abigail Lara Díaz

Este repositorio contiene mi portafolio profesional como **Senior QA Manual / QA Analyst**, enfocado en **pruebas funcionales**, **UAT** y **validación de procesos críticos** en sistemas **bancarios y de seguros**.

## Perfil profesional
Senior QA Manual con más de 5 años de experiencia participando en ciclos completos de pruebas dentro de entornos Agile/Scrum. Especializada en pruebas funcionales End-to-End, coordinación y ejecución de pruebas UAT, y validación de datos mediante SQL (Oracle). Experiencia trabajando con equipos de desarrollo y stakeholders en proyectos críticos y procesos de migración en el sector financiero.

## Áreas de especialización
- Pruebas funcionales End-to-End en sistemas bancarios y de seguros  
- User Acceptance Testing (UAT) con usuarios de negocio  
- Diseño de casos de prueba basados en requerimientos y riesgo  
- Pruebas de regresión y smoke testing previo a liberaciones  
- Documentación de pruebas y matrices de trazabilidad  
- Gestión y seguimiento de defectos  
- Validación y generación de datos de prueba con SQL (Oracle)  
- Metodologías Agile / Scrum  

## Automatización (como apoyo)
La automatización se utiliza como soporte al proceso de QA Manual:
- Cypress (automatización básica para regresión, smoke tests y end-to-end)
- Selenium WebDriver (mantenimiento y apoyo a pruebas)
